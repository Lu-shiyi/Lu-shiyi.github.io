Use Gridea and GitHub Pages to Build your Site

2021-11-11 Gridea

I will use xxx to replace your github name in this article.

  1. Download Gridea from here.
  2. Create a github repo named xxx.github.io.
    • replace user_name with your github user name. like Lu-shiyi.github.io.
    • set its visibility as public.
    • create repository.
  3. Generate a personal access token.
    • click your avator at the top right of your pages and enter Settings.
    • find Developer settings in the navigation on the right and click.
    • find Personal access tokens in the navigation on the right and click.
    • generate a new token just check the repo box.(your can set Note and expiration for it if you need).
    • save the token to your clipboad before you leave this page.
  4. Execute Gridea and connect your Github Pages.
    • click the button remote to fill in your infomation.
    • paste your new token into the token field and click the button save.
    • you can check the connection before you click the button synchronize.
    • click the button synchronize to push your changes to remote.
  5. Open a browser and type xxx.github.io, you will find your site is run successfully.
